# sudoku-solver
A basic sudoku solver, nothing much to see here

Basic approach is to infer what it can, then guess. If a guess is wrong, back-track. Will (eventually)
solve any sudoku you give it. The better the infer steps, the faster it is.

Inspired by Srini Devadas's [Programming for the Puzzled](https://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-s095-programming-for-the-puzzled-january-iap-2018/puzzle-8-you-wont-want-to-play-sudoku-again/)
